# Neverlose Lua API Docs

Документация по API чита Neverlose для разработки Lua-скриптов в CS:GO.

## Структура каталога

* docs/useful_information/quick_start.md — быстрое начало работы и базовая настройка скриптов.
* docs/useful_information/common_knowledge.md — общая информация о типах данных и среде выполнения.
* docs/useful_information/script_examples.md — примеры кода (вектора, цвета, ESP, ConVars).
* docs/documentation/events.md — список игровых и внутренних событий чита.
* docs/documentation/variables.md — описание глобальных объектов и библиотек.
* AI_PROMPT.md — подробный системный промпт (правила написания кода, особенности PUI, рендеринга и работы с энтити) для ИИ-ассистентов.

## Среда выполнения и технологии

Neverlose Lua работает на базе LuaJIT 2.1.0 (совместим с Lua 5.1). Поддерживаются библиотеки:
* FFI — прямой вызов C-функций и структур.
* BIT — побитовые операции.
* vector и color — встроенные объектно-ориентированные типы (создаются строго с маленькой буквы).

## Разработка интерфейса

Для создания пользовательского меню рекомендуется использовать библиотеку PUI (Perfect UI), которая импортируется через require("neverlose/pui"). Подробное описание методов PUI, автоматических зависимостей элементов (depend) и временной подмены значений (override) содержится в файле AI_PROMPT.md.
